# Vue JS project
The following packages have to be installed from this directory before you 
can run a development server of any kind. The docker containers come with the
packages installed, launching this project from another machine in a NodeJS
development server will require the following commands to be run from this
directory. 
```bash
cd app 
npm install -g @vue/cli
npm install -g @vue/cli-service
npm install -g @vue/compiler-sfc
npm install
npm run build
```
